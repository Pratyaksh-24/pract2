# Noir & Cream Style Guide

## Brand Archetype
The Sophisticated Rebel — dark, indulgent, unhurried.

## Color Palette

- **Obsidian (`#0A0A0A`)**: Base background. Deep black like cold glass in candlelight.
- **Aged Cream (`#F5ECD7`)**: Primary text and highlights. Warm cream of heavy poured dairy.
- **Kahlúa Brown (`#6B3A1F`)**: Accent. Deep coffee liqueur, warmth beneath the dark.
- **Amber Gold (`#C8A96E`)**: Glow. Hover states, CTAs, premium accents.
- **Condensation White (`rgba(255,255,255,0.06)`)**: Mist. Glassmorphism surfaces.

## Typography

- **Display**: Playfair Display (Italic serif for headlines)
- **Body**: DM Sans (Clean, modern, breathable)
- **Accent**: Cormorant Garamond (Thin weight for quotes and ingredient labels)

### Scale
- **H1**: clamp(4.5rem, 8vw, 6rem), italic, letter-spacing: -0.02em
- **H2**: 48px, weight 400
- **H3**: 28px, weight 500
- **Body**: 16px, line-height 1.8
- **Label**: 11px, uppercase, letter-spacing: 0.18em

# SEO Strategy Brief

## Core Meta Data
- **Meta Title**: Noir & Cream | The White Russian — Crafted with Intention
- **Meta Description**: The definitive guide to the White Russian cocktail. Technique, ingredients, and the ritual of indulgence. Three ingredients. One perfect drink.
- **OG Image**: Hero cream-pour shot — 1200×630px, dark, cinematic

## Best Practices Implemented
- **Title Tags**: Distinct `<title>` tags for each section, built hierarchically.
- **Heading Structure**: Single `<h1>` per page.
- **Semantic HTML**: `<section>`, `<nav>`, `<main>`, and `<footer>` elements used throughout.
- **Schema Markup**: Recipe schema provided for the `/the-craft` page to enhance search engine rich snippets.
- **Performance**: LCP target < 2.0s. All images served in next-gen formats (AVIF/WebP) and lazy-loaded below the fold.
